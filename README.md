### Pré Requisitos

- Git (https://git-scm.com/)
- Node (https://nodejs.org)

### Instalação

```bash
# Clone este repositório
$ git clone https://github.com/Jacoappolinario/Foodfy.git

# Vá para o repositório
$ cd Foodfy

# Instalar dependências
$ npm install

# Execute o aplicativo
$ npm start
```

### Tecnologias

- NodeJs
- Express
- Nodemon
- Nunjucks
